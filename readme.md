# MERN Boilerplate

MERN Boilerplate is a project made with the follow technologies:
- Express JS
- React JS
- MongoDB / Mongoose
- Typescript
- JWT
- NodeJs

## Set .env file
Example:
```bash
CLUSTER=false
NODE_ENV=development
PORT=8000
FRONTEND_URL=
```

## Installation

```bash
npm install
npm run dev
```
