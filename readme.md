# MERN Boilerplate

MERN Boilerplate is a project made with the follow technologies:
- Express JS
- React JS
- Redux
- MongoDB / Mongoose
- Typescript
- JWT
- NodeJs
## Installation
set .env file in root with this template:
```bash
CLUSTER=
NODE_ENV=
PORT=
FRONTEND_URL=
ACCESS_TOKEN_SECRET=
REFRESH_TOKEN_SECRET=
ACCESS_TOKEN_LIFE=
ADMIN_PASSWORD=
```
run to CLI:
```bash
npm install
npm run dev / npm run start
```
