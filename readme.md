# MERN Boilerplate

MERN Boilerplate is a project made with the follow technologies:
- Express JS
- React JS
- MongoDB / Mongoose
- Typescript
- JWT
- NodeJs
## Installation
set .env file in root with this template:
```bash
CLUSTER=
NODE_ENV=
PORT=
FRONTEND_URL=
```
run to CLI:
```bash
npm install
npm run dev
```
